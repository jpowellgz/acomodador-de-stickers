## Acomodador de stickers

Este es un programa simple para acomodar una imagen dentro de una hoja automáticamente. Para iniciar solo haz doble-clic sobre 'acomodador.exe'.
1. Para añadir una imagen, haz clic sobre el botón 'Elegir imagen'. La imagen aparecerá en el cuadro debajo del botón.
2. Debajo se encuentran las opciones de hoja. Existen opciones desde A0 hasta A10.
3. En cantidad, introduce la cantidad de stickers que quieres acomodar a lo ancho o a lo largo de la hoja.
4. En 'Acomodar' puedes elegir el número de stickers que van a pegarse a lo ancho o a lo largo.
5. Puedes elegir márgenes (en milímetros) para tu hoja. 
6. Existen dos formas de acomodo actualmente. Una para stickers rectangulares y otra optimizada para stickers circulares.
7. Después de elegir las opciones haz clic sobre Crear Hoja y se mostrará el resultado en la ventana.
8. Una vez tengas la hoja como quieras, puedes elegir el nombre y hacer clic en 'Guardar hoja'. Esto guardará la imagen en el mismo directorio que el programa 'acomodador.exe'.
